## ⚡ 3つの絶対的コア・バリュー（特徴）  
  
### 📱 1. モバイル・ファースト＆ランドスケープ・ハック  
* **縦画面（ROLL）：** ピアノロールをiPhone SEの画面に凝縮。右端へスクロール接近（残り50px）した瞬間、裏の可変配列データを1ミリ秒も遅延させずに1小節（16ステップ）ずつ自動でケツに結合していく「無限コンベア・ハック」を搭載。  
* **横画面（ST/GT）：** 端末を横に傾けた瞬間、ブラウザの入力モード（`inputmode="numeric"`）を強制クラック。余計なアルファベットキーを損切りし、画面に「数字専用テンキー」を強制起動させて本家さながらの爆速数値打ち込みを実現。  
  
### 🧬 2. 100%内製・依存性ゼロのクリーン知財  
外部の重たいサンプリング音源やOSSライブラリを1ビットも外部依存していません。Web Audio APIによる波形合成算数のみで駆動するため、ファイル1枚（数十KB）をローカルに落とすだけで、砂漠の真ん中（完全オフライン環境）でも爆速で曲が作れます。  
  
### 🎹 3. 決定論的SMF（MIDI）エディット＆保存  
打ち込んだデータは、独自に実装した可変長バイト（VLQ）関数を用いて、1クロックのズレもなく標準MIDIファイル（.mid）およびJSONデータへスマートに記帳・ファイル出力（デポジット）が可能です。
